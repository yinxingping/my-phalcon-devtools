# My Phalcon Devtools

## 与phalcon官方devtools的区别 
根据自身项目需求，对原开发工具做了若干小改动，并增加了新的项目模版
1. 小改动
    * 修正了即使新增的项目模版不需要css和js，都会自动创建相关目录的问题；
    * 出于安全和部署便利考虑，参考laravel增加了.env环境参数设置文件
2. 新增项目模版
    * baseapi

## 系统要求

* PHP >= 7.0
* Phalcon >= 3.2.0
* Composer
